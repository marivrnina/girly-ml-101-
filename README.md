# Girly ML 101

A collection of beginner-friendly machine learning projects, originally built for content on Instagram (@_girlyml_), applying core ML concepts to relatable, everyday scenarios.

## Overview
This repository holds a series of small, self-contained ML projects designed to make foundational concepts — regression, classification, exploratory data analysis — approachable by grounding them in familiar, real-life examples rather than abstract datasets.

## Tools and technologies
- Python
- pandas, numpy, scikit-learn
- Jupyter Notebook

## Methodology
Each project follows the same structure: data preprocessing, feature engineering, model selection, and evaluation — applied consistently across projects so the underlying ML concepts stay the throughline, regardless of the topic.
